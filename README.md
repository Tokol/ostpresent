# OSTP Pipe Roundness Inspection Presentation

This repository contains a Reveal.js presentation for the OSTP pipe roundness
inspection project.

Open `index.html` in a browser to view the presentation.

## Presentation Controls

- Press `F` to enter fullscreen presentation mode.
- Press `Esc` to exit fullscreen and return to the normal browser view.

Some slides include enhanced interactive visuals that are intended for
fullscreen presentation mode.
